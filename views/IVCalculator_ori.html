程序信息 ±

如何下载计算器

在浏览器窗口菜单中选择 文件 -> 另存为... 将网页保存到您的计算机中（请务必选择“网页，仅 HTML”方式保存）。该程序在脱机状态下仍然可用。

兼容性：

由于神奇宝贝X‧Y中一些神奇宝贝的种族值有变化，这个计算器并不适合之前的游戏，可以使用 适合第五世代及之前版本的个体值计算器。

关于

个体值 / 能力值 / 催醒计算器由 Legendary Pokémon 管理员 Hercules G. Papatheodorou (Arty2) 开发。该程序可以通过神奇宝贝的种族、性格、能力和基础得点数据来计算个体值。

该程序由 JavaScript 编写，具有跨操作系统的兼容性，只需一个文件和具有 JavaScript 功能的浏览器即可。该程序 应该可以 在任何操作系统的 Firefox, Chrome, Safari, Opera, Internet Explorer 等浏览器运行（任天堂 DSi 和 3DS 由于字库问题不支持中文版）。

个体值计算（简洁模式？）

* 星号标记的为可选项。

注意一定要记录基础得点，必要时可以使用降低基础得点树果，或使用超级训练系统的重置袋清除所有基础得点。

能力判断师位于奇南市神奇宝贝中心，可以告诉你神奇宝贝的全体能力和最高个体值评价，以及个体值为0的能力。在计算器中，可以使用 Ctrl/CMD 键选择（取消选择）多项能力。

使用神奇糖果将神奇宝贝提升到等级 20，就可以得到 100% 准确的能力值，在使用之前记得存档。

神奇宝贝模式：神奇宝贝的等级越高，计算出的个体值越准确。如果在较低的等级计算，请尽量多填写可选的内容。+等级 和 +基础得点 是指要计算出更精确的结果所需要的等级或基础得点。不同型态或百万进化的能力值也可以帮助计算。

队伍模式：可以一次计算多个神奇宝贝的个体值。

单行模式：可以计算多个神奇宝贝的个体并分别查看。

能力值计算

使用平均个体值，基础得点来准确计算出特定等级下神奇宝贝的能力值。

催醒计算

催醒的属性和威力取决于神奇宝贝的个体值，如果已知属性，可以更容易计算。

催醒超能力者位于百刻市神奇宝贝中心以东屋子里。

使用个体值分布（或平均个体值）可以得到所有可能的组合。

基础得点训练

基础得点追踪
取消选择选中的一行可以进入追踪模式，选择相应的道具、对手的神奇宝贝和数量来计算得到的基础得点。
基础得点计算
首先计算出神奇宝贝的能力值，然后改变第一行中的能力值，然后得出到达特定的能力值尚需的基础得点。
