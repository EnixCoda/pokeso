# 口袋搜·pokeso


用[AngularJS](https://github.com/angular/angular)、[Angular Material](https://github.com/angular/material)搭建的口袋妖怪工具网页。

发布方式仍不完善，目前使用grunt + Python发布。

操作(在项目根目录)：

1. pokeso $  grunt htmlmin
2. pokeso $  python merge\ views.py
3. pokeso $  grunt

可发布文件将集中在dist文件夹下。
